Roulette
=========

Author: James David Young

Product Description
-------------------------
This Roulette game is one of the seven games created for the Teesside casino project,
made to allow people to have fun gambling in unique and interesting ways, while losing
no real money!

Development Environment
------------------------
Python IDLE
Windows
Python version 3.11.2

Requirements
---------------
Python 3.6 and up
Pygame 2.6 and up

Install instructions
-----------------------
Download and install Python 3.12
Do pip 3 install pygame in powershell
Unzip the file in your directory

Usage instructions
---------------------
Run the Roulette Version 4.py file
